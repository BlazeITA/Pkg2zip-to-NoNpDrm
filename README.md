# Pkg2zip-Decrypted-Games-to-NoNpDrm-Zipped-Games
Batch script to merge PSVita base game, update and DLC into Vita3K-ready ZIP files. Includes WinRAR, 7-Zip and Windows built-in versions. WinRAR and 7-Zip are recommended for large games; the Windows built-in version has size limits and may fail on large archives. Read the README for more information and setup instructions.
